The [Audit Plugin](https://www.zen-cart.com/downloads.php?do=file&id=2261) helps you find and fix a number of database anomalies: 

- Product's master category does not exist
- Product not present in Products to Categories table
- Missing Product ID, Master Categories ID pair
- Master category ID 0
- Empty Categories
- Product with type 0
